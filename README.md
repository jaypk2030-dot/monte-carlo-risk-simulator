# 📊 Monte Carlo Risk Simulator

## 📌 Overview
Monte Carlo simulation project to model stock prices and estimate risk.

## 🚀 Features
- Stock price simulation
- Risk measurement (Volatility, VaR)
- Option pricing
- Data visualization

## 🛠 Tech Stack
- Python
- NumPy
- Matplotlib

## 📈 Output
- Histogram of prices
- Price paths
